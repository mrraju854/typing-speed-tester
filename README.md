# 🚀 Typing Speed Tester

A modern and fully responsive typing speed tester built using HTML, CSS, and JavaScript.

🔗 **Live Demo:** https://typing-speed-tester-raju.netlify.app/  
🔗 **GitHub Repo:** https://github.com/mrraju854/typing-speed-tester

---

## ✨ Features

- ⏱ 15-second challenge timer  
- ⚡ Real-time WPM (Words Per Minute) calculation  
- 🎯 Accuracy tracking  
- ✅ Auto-detect typing completion  
- 🎨 Premium dark UI design  
- 📱 Fully responsive (mobile, tablet, desktop)

---

## 🛠 Tech Stack

- HTML  
- CSS  
- JavaScript
